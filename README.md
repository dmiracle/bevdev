# Bevy learning project

## Plan

Build a first person 3d dungeon crawler game using rust and bevy.

## Methodology

This project is a combo learn-rust/learn-game-dev project. All the code in here is written using a keyboard and fingers like the heathen kings of old. AI tools are used to generate a learning pathway and guide what features to add next. The point of this is not generating as much code as possible, but instead to learn and avoid deskilling.
